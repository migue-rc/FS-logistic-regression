# Logistic Regression From Scratch

This project provides an educational, deep dive exploration of **logistic regression**, 
built entirely from scratch using Python and Numpy.
It is designed to bridge the gap between high level machine learning concepts and the fundamental mathematics that power them.

## Overview

The core of this project is an interactive [quarto notebook](https://migue-rc.github.io/FS-logistic-regression/) that guides you through the entire lifecycle of a *Logistic Regression* model.
Rather than relying on high level libraries like scikit-learn, I implemented the optimization process manually to provide a transparent look at how machines actually "learn".

![logregsigmoid](https://raw.githubusercontent.com/migue-rc/FS-logistic-regression/refs/heads/main/assets/hero.gif)
## Educational Focus

This serves as a foundational resource for anyone looking to understand:
- Binary classification and probability estimation.
- The sigmoid function and its role in logistic regression.
- How a loss function quantifies classification model performance.
- The iterative nature of optimization algorithms for classification.
- How to visualize decision boundaries and model convergence.

## Key Features

* **Mathematical Derivations**
* **Gradient Descent Intuition**
* **Interactive Visualizations**
